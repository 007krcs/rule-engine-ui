# Schemas

All schemas are versioned and validated before activation.

![Builder overview](/docs/builder-overview.png)

## UI schema (`UISchema`)

- `version`, `pageId`, `layout`, `components`
- component-level i18n + accessibility metadata
- optional bindings for `data.*` / `context.*`

## Flow schema (`FlowSchema`)

- `initialState`
- `states[stateId].on[event]`
- transition targets + actions + guards

For editing flow JSON in the current product surface, use `/docs/tutorial-flow-editor`.

## Rule set (`RuleSet`)

- JSON AST only (safe, deterministic)
- operators include logical, numeric, string, and date operators
- actions update data/context and emit traceable effects

## API mapping (`ApiMapping`)

- declarative request/response mapping
- no ad-hoc imperative API code in schema

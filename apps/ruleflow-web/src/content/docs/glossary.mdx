# Glossary

## adapterHint

A string like `material.input` or `company.currencyInput` that selects a UI adapter at render time.

## Component Registry

A registry of component metadata (display name, category, JSON Schema for props, default props, bindings supported).

## Config Package

A named configuration bundle containing versions of UI/flow/rules/API configuration.

## ExecutionContext

Immutable runtime metadata: tenantId, role, locale, device, permissions, feature flags.

## FlowSchema

A deterministic state machine defining states, transitions, guards, and actions.

## GitOps Bundle

A signed JSON export of the config store used to promote configs across environments.

## RuleSet

A list of rules expressed as a JSON AST (no code). Each rule has scope, condition, and actions.

## Trace

Structured output from runtime execution:
- flow transition details
- rules evaluation details
- API call details

## UISchema

A declarative UI description containing a layout tree and component list.

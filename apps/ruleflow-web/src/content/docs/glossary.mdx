# Glossary

![Docs overview](/docs/docs-overview.png)

## adapterHint

Prefix-based component selector such as `material.input` or `company.currencyInput`.

## Component Registry

Manifest of component metadata (display name, category, props schema, default props, supported bindings).

## Config Package

Named container of versioned bundles (UI, flow, rules, API mappings).

## ExecutionContext

Immutable runtime metadata: `tenantId`, role(s), locale, device, permissions, feature flags.

## FlowSchema

Deterministic state machine definition (states, transitions, guards, actions).

## GitOps Bundle

Signed JSON export/import payload used for moving config data between environments.

## RuleSet

JSON AST rule list with scope, condition, and actions.

## Trace

Runtime execution output containing flow, rules, and API details.

## UISchema

Declarative page description with layout tree and component list.

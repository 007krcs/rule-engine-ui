# Common Mistakes

![Docs overview](/docs/docs-overview.png)

## Builder / UISchema

- Missing `accessibility.ariaLabelKey`
- Missing or duplicate `accessibility.focusOrder`
- Layout references a missing component id
- `adapterHint` not registered in host app

## Flow / Rules

- Editing wrong `versionId` then expecting changes in Playground
- Operand paths missing `data.` or `context.` prefixes
- Expecting rules to change outcomes without pressing **Submit** in Playground

## Component Registry

- `adapterHint` not namespaced (`company.currencyInput` is good)
- malformed `propsSchema`
- duplicate `adapterHint` entries across manifests

## i18n / WCAG

- hard-coded labels instead of i18n keys
- missing keys in locale bundles
- skipping keyboard/focus verification after layout changes

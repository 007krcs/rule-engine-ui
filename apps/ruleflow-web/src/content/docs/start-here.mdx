## Quick reference

- `configId`: stable identifier for a config package
- `versionId`: immutable snapshot used by Builder, Rules Builder, and Playground
- `packageId`: internal package container ID used in Console

## Five-step first run

1. Create draft in **Get Started** (`Create draft` button).
2. Build and save UI in Builder.
3. Add and save a starter rule in Rules Builder.
4. Run Playground `Submit` and inspect `Trace` + `Explain`.
5. Promote in Console lifecycle (`Submit -> Approve -> Promote`).
